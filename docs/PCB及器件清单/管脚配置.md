### ICM20602

| 功能 |    管脚     |
| :--: | :---------: |
| SPC  | B13（SCK）  |
| SDI  | B15（MOSI） |
| SDO  | B14（MISO） |
|  CS  |     B11     |
| INT  |     B12     |
